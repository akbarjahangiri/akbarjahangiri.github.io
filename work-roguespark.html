3D Vampire Survivors

Unity Developer | RogueSpark Creations | Apr 2025 – Present

Project Overview

A 3D action-survivor game inspired by the popular Vampire Survivors genre, featuring intense combat, wave-based gameplay, and character progression systems.

My Role & Responsibilities

  • Player & Enemy Controllers: Developed responsive character movement systems and AI behavior for various enemy types
  • Ability Systems: Designed and implemented special abilities, upgrades, and character progression mechanics
  • Wave Management: Created procedural wave generation system that scales difficulty and spawns enemies dynamically
  • Performance Optimization: Implemented object pooling and event-driven architecture to handle hundreds of entities efficiently
  • Ability Synergies: Designed upgrade systems that create powerful combinations and enhance replayability
  • 3D Environment Setup: Contributed to level design, lighting setup, and visual polish using Unity URP and DOTween

Technical Achievements

  • Optimized game to handle 200+ entities simultaneously without performance drops
  • Implemented event-driven architecture for scalable ability and upgrade systems
  • Created procedural content generation for infinite replayability
  • Developed responsive combat system with satisfying visual and audio feedback

Technologies Used

  • Unity Engine (URP)
  • C# Programming
  • DOTween Animation
  • Object Pooling
  • Event-Driven Architecture
