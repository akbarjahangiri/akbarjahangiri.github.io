Gonbad Educational Game

Personal Project | Self-Employed | Unpublished

Project Overview

An interactive learning game designed to teach Farsi to children through playful storytelling and mini-games. Gonbad combines educational content with engaging gameplay to create an effective language learning experience for young learners.

My Role & Responsibilities

  • Game Design: Designed the overall game structure and learning progression system
  • Storytelling Integration: Created engaging narratives that incorporate Farsi language learning
  • Mini-Game Development: Developed various mini-games that reinforce language concepts
  • Child-Friendly UI: Designed intuitive interfaces suitable for children's interaction
  • Progress Tracking: Implemented systems to track learning progress and adapt difficulty
  • Cultural Integration: Incorporated Persian culture and traditions into the learning experience
  • Full Development: Handled all aspects of development from concept to implementation

Technical Achievements

  • Created an engaging educational platform that makes language learning fun
  • Developed child-appropriate UI/UX that encourages exploration and learning
  • Implemented adaptive learning systems that adjust to individual progress
  • Integrated storytelling with educational content for immersive learning
  • Designed culturally relevant content that connects with Persian heritage
  • Built a complete educational game from concept to playable prototype

Technologies Used

  • Unity Engine
  • C# Programming
  • Educational Game Design
  • Child UI/UX Design
  • Storytelling Systems
  • Progress Tracking

Project Impact

While unpublished, Gonbad represents my passion for creating educational content that bridges technology with cultural learning. The project demonstrates my ability to design engaging experiences that serve educational purposes while maintaining the fun and interactivity that makes learning effective for children.
